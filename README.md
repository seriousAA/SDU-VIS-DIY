使用说明：

别的没啥可说的，就是提示一下，右下角的平行坐标系是与鼠标的框选功能联动的

对哪一些人感兴趣，就直接鼠标左键框选就可以了，右下角的坐标系就会自动发生变化

还有就是Tag功能，将Tag选定到on上，然后再框选，就可以将人群标记成绿色，观察他们的运动

对了，Data功能必须要在平行坐标系上刷选后才能显示数据

网页布局我真的已经尽力了。。要是加载出来不对劲，可以缩放一下网页（手动滑稽）
